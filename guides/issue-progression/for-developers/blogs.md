# Writing Blog Posts 

We welcome blog posts to the **Hiero Website Repository** about this repository.

For example:
- New Releases
- New Community Calls
- Community Activity Updates
- Showcase Developers

Blog posts have to be submitted as pull requests to [Hiero Website Repository](https://github.com/hiero-ledger/hiero-website).

The [Hiero Website Blog Post Guide](https://github.com/hiero-ledger/hiero-website/blob/main/docs/blogs.md) documents how to set-up and create a blog post.

The Hiero website community will review your new blog post and publish it once approved.

Thank you!
